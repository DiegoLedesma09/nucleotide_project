# 🧬 Analizador de Secuencias de ADN

Scripts básicos en Python para el procesamiento y validación de secuencias de ADN.

Este repositorio incluye funcionalidades para:

- ✅ Limpieza de secuencias
- 🔍 Validación de caracteres inválidos
- 🔢 Conteo de nucleótidos (A, T, G, C)
- 📏 Cálculo de longitud de secuencia
- 📊 Cálculo del contenido GC

Este proyecto está pensado como ejercicio introductorio en bioinformática utilizando Python.
